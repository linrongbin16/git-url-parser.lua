Pure Lua implemented git URL parsing library, e.g. the output of `git remote get-url origin`.

## Features

* [ ] Single file & zero dependency.
* [ ] Support lua >= 5.1, luajit >= 2.0.0.
* [ ] Compatible with RFC 3689.
